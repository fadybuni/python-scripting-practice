# Python Text Processing Scripts

A small collection of beginner-friendly Python scripts for practicing
file I/O, string processing, and basic scripting concepts.

## Scripts

### keyword_counter.py
Scans a text file for a user-provided keyword and reports how many times it appears.

### log_error_finder.py
Analyzes a `.log` file and counts how many lines contain error messages.

### word_frequency.py
Counts how often each word appears in a text file and outputs the most common words.

### login_attempt_analyzer.py
Parses a login log file and summarizes successful and failed login attempts.

## Concepts Practiced
- File reading and writing
- Dictionaries and counting
- Sorting data
- User input handling
- Basic automation scripts
